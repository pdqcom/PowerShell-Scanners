# Windows Firewall Rules

## Instructions

[How to use this repository](../../README.md)

## Description

This scanner returns Windows Firewall Rule information. Requires Windows Server 2012/Windows 8 or higher.

## Compatibility

* **PDQ Inventory**: Yes
* **PDQ Connect**: Yes

## Parameters

```powershell
-Enabled
```

Returns only enabled rules if used.
